üîå API Endpoints

GET /health
Server health check - returns status and uptime
POST /api/entry
Index an elfeed entry - accepts JSON with title, entry_id, summary, content, md5. Automatically generates embeddings and stores in database.
POST /api/search/semantic
Semantic search by meaning - accepts query (string), title (string), and optional model. Returns ranked results with similarity scores, entry titles.
GET /api/stats/
Get comprehensive database statistics including entry counts, embeddings and database size

üöÄ Getting Started

1. Quick Start from Emacs

Open the main menu with:

M-x org-db or H-v

2. Search Commands

  • v - Semantic search - Find content by meaning using embeddings

3. Indexing Commands

  • u - Update current entry
  • U - Update all entries
  • r - Reindex database

5. First Time Setup

Add to your Emacs config:

(add-to-list 'load-path "/path/to/elfeed-db-summary/elisp")
(require 'elfeed-db-summary)

;; Bind the menu to a convenient key
(global-set-key (kbd "H-v") 'elfeed-db-summary-menu)

;; Auto-start server when Emacs starts (optional)
(setq elfeed-db-summary-auto-start-server t)
